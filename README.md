# Trading Simulator
元本、保有期間、ロスカット、利確ラインを設定して公開している日経全銘柄に対してシミュレーションをしてグラフに描画する

## Client
1. npm -w packages/client dev

## Server

1. docker compose up -d
2. npm -w packages/server start
